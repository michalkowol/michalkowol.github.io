How to configure LB-LINK

2017-01-21
howto random network

Overview

Do you want to change WiFi to regular cable connection. Or you need cheap WiFI amplifier? This post is for you!

You can by LB-LINK BL-MP01 for only 13$ (52 PLN). It is really cheap and it does it work.

Repeater (client mode)

  1. If you want to change WiFi to regular cable connection you need to switch device to Repeater mode.
  2. Connect power and Ethernet cable.
  3. Make sure power mode is on.
  4. Turn off your WiFi adapter (please remember to turn it on after configuration is done).
  5. Open configuration page: http://192.168.16.254/ in Internet Explorer (it does not work correctly on Chrome).
  6. Enter credentials (default - admin:admin)
  7. Go to Wireless tab and then open site_survey sub-tab.
  8. Select wireless network to connect and click Connect button. Enter your WiFi password.
  9. After successful connection you should see handshake icon.
  10. Test you connection - http://www.google.com/
  11. Turn on your WiFi adapter.
  12. Connect device you want to use using Ethernet cable (Console, Cable Box device etc., old computer).

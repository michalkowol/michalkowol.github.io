Encrypt zip files on MacOS

2016-12-16
random
  1. Press cmd + space, type ‘terminal’ and press enter.
  2. Type cd (with space) and do not press enter.
  3. Drag-and-drop folder you want to zip to terminal window and press enter.
  4. Type cd ...
  5. Type zip -er <filename.zip> <dir-you-want-to-encrypt> in example zip -er aurator.zip aurator/
  6. Type password two times in terminal window (password characters will not be visible).
  7. Your encrypted file is created!

Ambiguity

The main backward of this approach is possible ambiguity in sectionParameters and editorialParameters

Example:

{
  "sectionCode": "ambiguity-example",
  "sectionParameters": {
    "title": "foo"
  },
  "editorialParameters": {
    "title": "bar"
  }
}
